# COVID-19 Data Analysis & Prediction

This project analyzes global COVID-19 data, visualizes trends, and builds a simple predictive model using Python.

## Steps:
1. Load and clean the dataset
2. Visualize total cases over time
3. Identify top 10 countries by cases
4. Build a linear regression model to predict future cases

## Tools:
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn
- Jupyter Notebook

## Findings:
- Countries with highest cases: USA, India, Brazil
- Linear regression provides rough trend prediction
- Visualizations show growth and decline phases

## How to Run:
1. Install dependencies: `pip install -r requirements.txt`
2. Open Jupyter notebook: `notebooks/covid_analysis.ipynb`
3. Run all cells to generate plots and predictions
